A page without considered typography is a speech delivered by a man in a borrowed suit — legible, perhaps, but nobody leans in. Typography is not decoration; it is 95% of what the user experiences and roughly 100% of what they remember.

## The five jobs
Hierarchy, tone, rhythm, trust, conversion. Each earned through restraint.

## The pairing rule
One distinctive display face + one neutral text face. Good modern pairings: Fraunces + Inter; Instrument Serif + Söhne; Migra + Mona Sans; Playfair Display + IBM Plex Sans; or General Sans variable for the whole site.

## Modular scale (approximate)
- H1: 56–80 desktop / 36–44 mobile
- H2: 36–44 / 28–32
- H3: 24–28 / 22–24
- Body: 17–19 / 17–18

## The three numbers
- Body line height 1.5–1.65
- Measure 60–75 characters per line
- Display line height 1.05–1.15

## Mobile
17px minimum body, loosen line height on mobile, avoid long all-caps headlines, generous padding around inline links.

## Variable fonts
Ship one variable file. Stop shipping five static WOFF2s named like 2013 family photos.

## The ten sins
Inter 16 everywhere, lorem ipsum headlines left behind, justified desktop body, centered multi-line body, six weights, display face as body, default tracking on 72px, `#999` on white, underlines removed, two display faces in the hero.

## Verdict
Typography is not decoration; it is the page — and a site that writes its letters with intention is a site the reader agrees to believe.
